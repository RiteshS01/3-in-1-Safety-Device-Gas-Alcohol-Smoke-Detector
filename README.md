# 3-in-1-Safety-Device-Gas-Alcohol-Smoke-Detector
The 3-in-1 Safety Device is a compact and cost-effective safety monitoring system designed to detect hazardous gases, smoke, and alcohol vapors using a single sensing unit. The project aims to improve safety in homes, industries, laboratories, public spaces, and vehicles by providing real-time hazard detection and instant alerts.

Traditional safety systems often require separate devices for gas leak detection, smoke detection, and alcohol monitoring. This project integrates all three functionalities into a single portable device, reducing cost, power consumption, and installation complexity while improving overall safety.

🎯 Problem Statement
Gas leaks, fire incidents, and alcohol-related hazards are among the leading causes of accidents in residential and industrial environments. Existing solutions generally focus on detecting only one type of hazard, requiring multiple devices and increasing maintenance costs.

This project addresses the need for a unified safety solution capable of detecting:
*Smoke generated by fire incidents
*Harmful or combustible gases
*Alcohol vapors in the surrounding environment

The device provides early warning alerts, allowing users to take preventive action before a situation becomes dangerous.

🚀 Objectives
*Detect smoke, gas leaks, and alcohol vapors using a single device.
*Provide real-time monitoring and hazard alerts.
**Reduce installation and maintenance costs compared to multiple detectors.
*Create a portable and battery-powered safety solution.
*Demonstrate sensor fusion in embedded safety applications.

⚙️ Working Principle
The system is based on the MQ-2 Gas Sensor, which is capable of sensing smoke, LPG, methane, and alcohol vapors. When the concentration of any hazardous substance exceeds a predefined threshold:
*The MQ-2 sensor detects the presence of smoke, gas, or alcohol.
*The sensor output activates a BC557 transistor switching circuit.
*The transistor drives an LED indicator and buzzer.
*The LED glows and the buzzer sounds an alarm to alert nearby users.
*The system continues monitoring until the hazardous condition disappears.

🛠️ Hardware Components
Component	Purpose
1.MQ-2 Gas Sensor	Detects smoke, gas, and alcohol vapors
2.BC557 Transistor	Signal amplification and switching
3.LED	Visual indication
4.Passive Buzzer	Audible alarm
5.TP4056 Charging Module	Battery charging and protection
6.3.7V Lithium-Ion Battery	Portable power source
7.Resistors (100Ω, 470Ω)	Current limiting
8.Switch	Power control

🏗️ System Architecture
Power Supply Unit
3.7V Lithium-Ion rechargeable battery
TP4056 charging module for safe charging
Detection Unit
MQ-2 sensor continuously monitors air quality
Detects smoke, combustible gases, and alcohol vapors
Alert Unit
LED for visual alerts
Buzzer for audible warnings
